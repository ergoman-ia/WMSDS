▮ 포토룰러 사진 속 물건 크기 재기

기준 미설정
사진을 끌어다 놓거나 클릭해서 선택
크기를 아는 물건(카드, A4 용지, 자 등)이
함께 찍힌 사진이면 더 정확해요.

사용 방법

1. 사진 업로드 → 2. 기준 지정 → 3. 실제 크기(cm) 입력 → 4. 측정 모드로 물건 위를 드래그

모드

크기를 아는 물건 위에 드래그해 노란 기준선을 그려주세요. (신용카드 가로 8.56 cm, A4 세로 29.7 cm)
크기를 아는 직사각형 물체(A4, 카드, 책, 타일 등)의 네 모서리를 좌상단부터 시계 방향으로 클릭하세요. (0/4)

측정 결과

    측정한 항목이 여기에 표시됩니다.

    AI 물건 분석 — Gemini (선택)

    Google AI Studio(aistudio.google.com)에서 발급한 API 키를 입력하고, 기준을 설정한 뒤 [분석]을 누르세요. 키는 저장되지 않고 이 화면에서만 사용됩니다.

    정확도 팁

    · 사진이 비스듬히 찍혔다면 반드시 [사각형 기준]을 쓰세요. 네 모서리로 기울어진 평면을 정면으로 펴서(원근 보정) 계산합니다.
    · 원근 보정은 기준 사각형과 같은 평면(같은 바닥, 같은 벽, 같은 책상면) 위 물건에만 정확합니다. 평면에서 튀어나온 높이 방향은 별도 기준이 필요해요.
    · 직선 기준은 정면에서 찍은 사진에만 적합합니다.
    측정값은 사진 기준 근사치입니다 · 정밀 치수가 필요하면 실제 자로 확인하세요
